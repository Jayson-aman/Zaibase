# 問屋ヒアリング トークスクリプト（資材EC・ステップ1）

> 大阪・神奈川の建材問屋10社に電話。ユーザーゼロでも実施可。  
> 目的：**ドロップシッピング（モデルB）** と **グループ購買（モデルC）** の関心確認。

---

## 事前準備（1分）

- 自己紹介：「建設業マッチングアプリ **Zaibase（ザイベース）** の〇〇です」
- 伝えたいこと：**在庫を持たない仲介**・**職人のまとめ買い**・**石油系資材優先**

---

## オープニング（30秒）

```
お世話になっております。建設業向けプラットフォーム Zaibase の〇〇と申します。
突然のお電話失礼いたします。

職人・一人親方がスマホから資材を注文できるECを準備しており、
御社のような問屋様と「在庫を持たず、注文後に御社から直送」する
ドロップシッピングの提携を検討しています。

2〜3分だけお時間いただけますでしょうか。
```

---

## 本題（3分）

### ① 関心の確認

```
御社では、オンライン経由（楽天・自社EC等）での個人・小規模工事業者向け
販売はいかがでしょうか。今後、第三者マーケットへの出品にご関心はありますか。
```

### ② モデルB（ドロップシッピング）

```
Zaibase側は在庫を一切持ちません。職人が注文 → 御社が直接配送 → 
Zaibaseは仲介手数料（目安5〜15%）のみ、という形を想定しています。
```

### ③ モデルC（グループ購買）

```
職人10人分をまとめて1発注にし、大量購入割引を職人に還元する
「共同仕入れ」も計画しています。御社にとってもロットが大きくなります。
```

### ④ 優先カテゴリ

```
最初のターゲットは石油系資材です。
シーリング・コーキング、塗料・防水材を優先したいと考えています。
```
