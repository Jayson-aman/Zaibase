# Zaibase 全サービス 法令対応・必須ページ作成指示書（追補）

## 全体方針

Wix 新ドメイン公開前に、サービスごとに関係法令を確認し、
**利用規約・特定商取引法に基づく表示・プライバシーポリシー** を整備する。
「全部対応」＝以下のチェックリストをすべて満たすこと。

## 1. サービス別 関係法令チェックリスト

### A. 法律相談（AIシミュレーション）

| 法令 | 要点 |
|------|------|
| 弁護士法第72条 | 非弁護士の有償法律相談・解決請負禁止 → 弁護士へのつなぎ・AIシミュレーション SaaS に限定 |
| 特定商取引法 | 月額サブスク＝継続的役務提供 |
| 個人情報保護法 | 相談内容・利用ログの取扱い |

### B. 受験サポート

| 法令 | 要点 | サイトで必要な対応 |
|------|------|-------------------|
| 特定商取引法（継続的役務） | 月額・年額の継続課金は「継続的役務提供」 | 契約期間・更新・解約・中途解約の条件を明記 |
| 景品表示法 | 過大な合格保証・「必ず合格」等は優良誤認 | 合格保証・合格率の断定表現禁止。「サポート」「学習支援」と表現 |
| 個人情報保護法 | 氏名・成績・志望校等 | プライバシーポリシー必須 |

NG例：「必ず合格」「合格率○○%」「国家資格取得を保証」「法律相談」と混同される表現
OK例：「学習計画のサポート」「過去問演習の提供」「結果を保証するものではありません」

### C. 建設・工事

| 法令 | 要点 | サイトで必要な対応 |
|------|------|-------------------|
| 建設業法 | 建設業許可がないと建設工事の請負・仲介を業として行えない | 許可番号・許可行政庁は許可取得後のみ表記。未取得なら工事請負の募集・仲介表現禁止 |
| 宅建業法 | 不動産仲介がある場合のみ | 該当しなければ記載不要 |
| 特定商取引法 | 継続課金・着手金等がある場合 | 役務の内容・代金・解除条件を明記 |
| 個人情報保護法 | 現場写真・住所・契約情報 | プライバシーポリシー必須 |

NG（許可なしの場合）：「工事請負」「施工代行」「元請け・下請けのマッチング（報酬あり）」、建設業者であるかのような表示
OK（許可なしでも可）：情報共有・帳票・進捗管理SaaS（建設工事の請負に該当しない旨を明記）、「提携施工業者のご紹介（連絡先の情報提供のみ）」

許可取得済みの場合の必須表記：許可番号、許可行政庁名、代表者名・事業所所在地（特商法と整合）

## 2. 必須ページ一覧（共通）

| ページ | 必須度 |
|--------|--------|
| 特定商取引法に基づく表示 | 有償・通信販売・継続課金がある場合必須 |
| プライバシーポリシー | 必須（個人情報を1件でも取得する場合） |
| 利用規約 | 強く推奨（実質必須） |
| 提携弁護士（法律系のみ） | 法律系のみ。紹介に限る旨の免責付き |

## 3. 特定商取引法 記載必須項目

共通：販売業者／運営統括責任者／所在地／電話番号／メールアドレス／販売価格／商品代金以外の必要料金／支払方法・時期／商品の引渡し時期／返品・キャンセル

継続的役務（月額サブスク）追加必須：契約期間／更新の有無・手続／解約・中途解約の条件／中途解約時の精算・違約金（なしの場合も明記）／役務の内容

注意：Stripe / RevenueCat 等の決済表示と文言を一致させる（プラン名・金額・解約方法）。

## 4. プライバシーポリシー 必須セクション

事業者名・連絡先／取得する個人情報の種類／利用目的／第三者提供（Stripe, RevenueCat, Google, クラウドホスト等）／委託（GCP/Firebase等）／保管期間・削除／ユーザーの権利／Cookie・アクセス解析／改定手続／未成年（該当する場合）
