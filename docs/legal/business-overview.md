## 1. 事業の概要

**Zaibase建設**は、建設職人・一人親方と発注者（元請・工務店）を繋ぐ、建設業特化のプラットフォームサービスです。

### 対象ユーザー
- **職人（受注者）**：一人親方・職人・下請け業者・外国人技能実習生・特定技能労働者
- **発注者**：元請業者・工務店・現場監督

### 現在の収益モデル
月額サブスクリプション（Stripe課金）

| プラン | 月額 | 年額 |
|---|---|---|
| フリー | 無料 | 無料 |
| スターター | ¥1,980 | ¥19,800 |
| プロ | ¥5,480 | ¥54,800 |
| チーム（5名） | ¥14,800 | ¥148,000 |

## 2. プラットフォームの基本的な立場

- **Zaibaseは当事者間の契約に関与しない**。職人と発注者が直接請負契約を結ぶ。
- **決済代行を原則としない**。報酬は当事者間で直接振込。エスクロー機能はあるがStripe Connect経由（β）。
- **法的判断を提供しない**。法令情報・チェックリストの提供にとどまり「弁護士にご相談ください」と明示。
- **職業紹介事業者ではない（現時点）**。マッチング手数料課金は停止中（有料職業紹介事業者許可申請中）。

## 6. 現在の法令対応状況

| 法令 | 対応内容 |
|---|---|
| 特定商取引法 | アプリ内に特商法表記画面あり |
| 個人情報保護法 | プライバシーポリシー整備済み。顔認証は同意取得・撤回機能あり。 |
| 建設業法 | 見積書・施工体制台帳・工期管理・コンプライアンス各画面で条文準拠を明示 |
| 職業安定法 | 有料マッチング停止中・免責事項明示済み・許可申請手続き中 |
| 弁護士法第72条 | 法的判断不提供の免責明示・弁護士への相談を推奨する文言あり |
| 社会保険労務士法第27条 | 書類補助のみ・申請代理なしを明示 |
| 資金決済法 | Stripe Connect利用（Stripe側がライセンス対応）。Zaibaseは資金移動業者ではない。 |
| 下請代金支払遅延等防止法 | 下請コンプライアンス画面で法令遵守チェックリストを提供 |
