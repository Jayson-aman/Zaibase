## 郵送先・手続き

### 送り先

```

```

### 同封するもの

| 同封物 | 枚数 | 備考 |
|---|---|---|
| 商標登録願（上記） | 1部 | A4・片面・押印必須 |
| 特許印紙 | 1枚 | ¥7,200（2区分分） |

### 特許印紙の買い方

- **特許庁内の売店**（直接持参の場合）
- **東京・大阪の法務局** → 窓口で「特許印紙を¥7,200分ください」
- **郵便局**（一部の局）→ 収入印紙と混同しないよう注意。**特許印紙**と明示する

> 特許印紙は収入印紙とは別物。コンビニでは買えない。

### 費用の内訳

| 区分 | 出願料（書面出願） |
|---|---|
| 第37類 | ¥3,600 |
| 第42類 | ¥3,600 |
| **合計（特許印紙）** | **¥7,200** |

> 書面出願は電子出願より1区分あたり¥200高い（¥3,600 vs ¥3,400）

### 登録料（査定後・約1年後）

- 2区分 × ¥17,200 ＝ **¥34,400**（査定後30日以内に納付）

## 出願後の流れ

```
郵送
  ↓（約2週間）
出願番号の通知ハガキが届く
  ↓（約8〜12ヶ月）
審査結果（登録査定 または 拒絶理由通知）
  ↓ 登録査定の場合
登録料 ¥34,400 を納付
  ↓（約2週間）
登録証が届く → 商標権発生（10年間有効）
```

## 拒絶理由通知が来た場合

- 意見書・補正書を提出すれば再審査になる
- 「JINZAI BASE」との混同を指摘された場合：
  - 称呼の違い（ジンザイベース vs ザイベース）を意見書で主張
  - 外観・観念の違いも添付
  - この作業だけ弁理士に依頼しても ¥3〜8万程度
