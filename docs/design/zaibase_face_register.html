顔認証データの利用について

現場への入退場を顔認証で自動記録します。就業データはCCUSに自動連携され、手入力が不要になります。

✓ 顔画像はAWS Rekognitionにのみ保存されます
✓ Zaibaseサーバーには画像を保存しません
✓ いつでも削除・同意撤回できます
! 顔認証データは要配慮個人情報です（個人情報保護法）

上記の内容を確認し、顔認証データ（生体情報）の収集・利用・AWS Rekognitionへの提供について同意します。同意はいつでも設定画面から撤回できます。
