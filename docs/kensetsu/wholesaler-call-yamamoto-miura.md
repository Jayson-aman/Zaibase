# 問屋電話スクリプト（即日使用可）

> 生成日: 2026-06-09  
> アプリ内でも **マイページ → 💰 会員価格・交渉 → 交渉スクリプト** で再生成・コピーできます。  
> **契約書テンプレート（締結用）:** [`docs/contracts/zaibase_dropship_contract.docx`](./contracts/zaibase_dropship_contract.docx)（§27・弁護士確認前ドラフト）

共通前提:
- 登録職人数: **100名**（神奈川・大阪中心）
- Zaibase手数料: 10%（在庫リスクなし・ドロップシッピング想定）
- 担当者名・電話番号は発信前に差し替えてください

---

## 1. 山本政清商店

**希望条件:** 会員 7掛け / バルク 5.5掛け  
**月間見込み購買額:** 約300万円

### 📞 電話・訪問スクリプト

```
お忙しいところ恐れ入ります。
「Zaibase（ザイベース）」という建設職人向けプラットフォームを運営しております、
（担当者名）と申します。

現在、弊社のプラットフォームには100名の職人・建設業者が登録しており、
山本政清商店様の商品に対して月間約300万円の購買需要があると試算しています。

本日ご連絡した目的は、弊社会員向けの特別価格でのご供給についてご相談したいためです。

具体的には、
・会員向け価格：定価の7掛け程度
・10名以上のグループ購買時：定価の5.5掛け程度
でのご対応が可能かどうかお聞きしたいのですが、
ご担当者様とお打ち合わせのお時間をいただけますでしょうか？

弊社からは
・月間の発注量をコミットできること
・一括発注・一括決済で業務コストを削減できること
・地域の職人へのブランド認知向上
といったメリットをご提供できます。

30分程度でも構いません。ご都合のよい日程を教えていただけますか？
```

### ✉️ 初回メール（山本政清商店向け）

**件名:** 建設職人向けプラットフォーム「Zaibase」からのご提案（山本政清商店 様）

```

ご担当者様

はじめてご連絡申し上げます。
一般社団法人Zaibase（ザイベース）の（担当者名）と申します。

現在100名の職人・建設業者が登録しています。

このたび、登録会員への資材供給パートナーとして、
貴社との取引を検討いただけないかとご連絡いたしました。

▼ 弊社の特徴
・登録職人数：100名（神奈川・大阪エリア中心）
・月間資材購買見込み：約300万円
・グループ購買（10名以上まとめ発注）の仕組みあり
・一括発注・一括決済で貴社の業務負荷を軽減

▼ ご提案内容
会員向け価格（掛け率7掛け程度）・バルク5.5掛けでのご供給を
検討いただけますでしょうか。

まずは30分程度のお打ち合わせの機会をいただけますと幸いです。

Zaibase 担当者
Email：info@zaibase.jp
```

### 電話前チェックリスト（山本政清）

- [ ] シーリング・タイル資材の主力カテゴリを1つ具体名で言える
- [ ] 翌日配送エリア（大阪・神奈川）を確認する質問を用意
- [ ] ドロップシッピング（在庫を持たない）可否を確認
- [ ] 価格表CSVまたはAPI連携の可否を確認
- [ ] 担当者メールアドレスを取得

## 電話後の記録（Zaibaseアプリで更新）

交渉が進んだら **問屋交渉管理** タブで状況を更新してください。

| 問屋 | 交渉結果の記録例 |
|---|---|
| 山本政清 | 掛け率確定・契約日・月取引見込み |
| ミウラ | 掛け率確定・配送条件・担当者名 |

確定した掛け率は **会員別価格** タブの商品価格表に反映します（電話後タスク）。
